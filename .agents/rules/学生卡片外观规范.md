# 学生卡片外观规范

## 核心原则

学生卡片在**所有位置**（座位图、候选栏、对话框等）必须保持**完全一致**的外观和长宽比。

## 标准尺寸

### 基准尺寸（桌面端，>1366px 宽度）
- **宽度**: 120px
- **高度**: 80px
- **长宽比**: 3:2（横向卡片，宽度大于高度）
- **边框**: 2px solid
- **圆角**: 12px

### 响应式断点

| 断点条件 | 宽度 | 高度 | 圆角 |
|---------|------|------|------|
| 默认（>1366px） | 120px | 80px | 12px |
| 1025px - 1366px | 102px | 68px | 10px |
| 高度 ≤820px 且 宽度 >1025px | 96px | 64px | 9px |
| ≤768px | 105px | 70px | 10px |
| ≤480px | 82.5px | 55px | 8px |

## 布局容器规范

### 座位图（SeatChart）
- 使用 CSS Grid 布局
- 每个座位占据一个 grid cell
- Grid 列宽: `120px`（与卡片宽度一致）
- Grid 行高: `80px`（与卡片高度一致）
- 间距: `12px`

### 候选栏（StudentList .student-items）
- 使用 CSS Grid 布局
- `grid-template-columns: repeat(auto-fill, 120px)`
- `grid-auto-rows: 80px`
- 间距: `12px`

## 内部元素规范

### 学生姓名
- 字体大小: 20px（默认）
- 字重: 600
- 颜色: #333
- 行高: 1.3

### 学号
- 字体大小: 12px（默认）
- 字重: 700
- 颜色: #23587b
- 背景: white
- 内边距: 2px 10px
- 圆角: 6px

### 标签（颜色点模式）
- 尺寸: 6px × 6px
- 形状: 圆形
- 间距: 3px

### 标签（文字模式）
- 字体大小: 9px
- 字重: 600
- 颜色: white
- 内边距: 1px 4px
- 圆角: 3px

## 实现要点

1. **禁止使用 aspect-ratio**：由于容器宽度固定，使用 aspect-ratio 会导致高度计算错误
2. **同时指定宽高**：卡片必须同时指定 width 和 height，不依赖自动计算
3. **容器与卡片尺寸一致**：Grid 的列宽/行高必须与卡片的 width/height 完全匹配
4. **响应式同步**：所有断点的尺寸调整必须在 SeatItem、CandidateItem 和容器中同步
