## 安全特性

### 加密强度
- ✅ AES-GCM 256-bit（行业标准）
- ✅ PBKDF2 密钥派生（100,000 迭代）
- ✅ 随机 IV（每次加密不同）
- ✅ 认证加密（防篡改）

### 密钥管理
- **已登录 SCE**: 使用用户 Token 派生密钥（安全性高）
- **未登录**: 使用通用密钥（安全性低，已在 UI 中警告）

### Cookie 安全
- `SameSite=Lax`: 防止 CSRF
- `Secure`: 仅 HTTPS 传输（生产环境）
- `Max-Age=7天`: 自动过期

## 用户体验

### 默认行为
- 默认**不勾选**"记住密码"
- 用户需要主动选择

### 安全提示
- 明确显示加密存储警告

### 便利性
- 勾选后无需每次输入密码

- 可随时清除已保存密码

## 风险与缓解

### 已知风险
1. **XSS 攻击**: 前端加密无法防止 XSS 获取密钥
   - 缓解: 严格的 XSS 防护措施
   
2. **通用密钥泄露**: 未登录用户使用硬编码密钥
   - 缓解: UI 警告 + 推荐登录 SCE

3. **中间人攻击**: HTTP 环境下 Cookie 可能被截获
   - 缓解: 生产环境强制 HTTPS

### 最佳实践
- 登录 SCE 账号后使用
- 使用 WebDAV 应用专用密码
- 避免在公共设备上勾选
- 定期更换密码
