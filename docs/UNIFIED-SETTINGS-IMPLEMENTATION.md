# 统一设置菜单实现总结

## 完成时间
2026-04-26

## 实现概述

已完成统一设置菜单的实现，包含全局设置和工作区设置两大类，共 8 个功能面板。

## 全局设置（3个面板）

### 1. 云端同步面板 (SyncPanel.vue)
- **功能**：与独立同步设置对话框完全一致
- **特性**：
  - WebDAV 服务器配置（地址、用户名、密码）
  - 自动备份模式开关
  - 默认读写目标选择（SCE 云/WebDAV）
  - 实时同步状态显示
  - 验证并保存到账号数据库
  - 支持清空配置
- **数据源**：直接使用 `useAuth` 和 `useWebDav` composables

### 2. 界面偏好面板 (UIPanel.vue)
- **功能**：自定义界面外观
- **设置项**：
  - 语言选择（简体中文/English）- 多语言功能搁置
  - 主题色自定义（颜色选择器）
  - 默认缩放比例（50%-200%）
  - 启用动画效果
- **已移除**：紧凑模式（无效功能）
- **数据源**：通过 props 接收 `settings.ui`

### 3. 编辑器行为面板 (EditorPanel.vue)
- **功能**：配置编辑器默认行为
- **设置项**：
  - 自动保存间隔（秒）
  - 撤销历史大小（10-100步）
  - 拖拽灵敏度（0.5x-2.0x）
  - 双击座位行为（编辑/分配）
- **已移除**：默认文件格式（无效功能）
- **数据源**：通过 props 接收 `settings.editor`

## 工作区设置（5个面板）

### 1. 工作区信息面板 (WorkspaceInfoPanel.vue)
- **功能**：工作区概览和快捷入口
- **统计信息**：
  - 学生总数、已分配座位、空座位
  - 标签数量、座位总数、规则数量
- **快捷入口**：
  - 编辑学生名单（打开 StudentRosterDialog）
  - 标签设置（打开 TagSettingsDialog）
- **特点**：带图标的卡片式统计，悬停动画效果

### 2. 座位表配置面板 (SeatConfigPanel.vue)
- **功能**：基本座位表布局配置
- **基本配置**：
  - 大组数量、每组列数、每列座位数
  - 错位距离
  - 讲台位置（底部/顶部）
  - 座位对齐方式（对齐底部/顶部）
- **高级配置入口**：
  - 打开侧边栏"编辑"标签（tab 2）进行各组独立配置
  - 点击后自动关闭设置对话框
- **数据源**：直接操作 `seatConfig`

### 3. 选区轮换面板 (RotationPanel.vue)
- **功能**：选区轮换系统说明和入口
- **功能介绍**：
  - 循环模式（A→B→C→A）
  - 互换模式（A↔B）
- **使用说明**：
  - 创建轮换组和选区
  - 框选座位
  - 手动触发轮换
- **入口**：打开侧边栏"排位"标签（tab 3）
- **点击后自动关闭设置对话框**
- **数据源**：显示 `rotGroups` 数量

### 4. 智能排位面板 (AssignmentPanel.vue)
- **功能**：智能排位算法说明和入口
- **算法特性**：
  - 模拟退火算法
  - 多维约束满足
  - 智能优化（优先处理违规学生）
- **功能说明**：
  - 自动处理必需规则
  - 支持区域标签约束
  - 动态温度调整
  - 实时进度显示
- **入口**：打开侧边栏"排位"标签（tab 3）
- **点击后自动关闭设置对话框**
- **数据源**：显示 `rules` 数量

### 5. 导出配置面板 (ExportPanel.vue)
- **功能**：导出系统概览和入口
- **导出类型**：
  - 图片导出（PNG/JPEG）- Canvas 渲染，A4 优化
  - Excel 导出（XLSX）- 带样式和合并单元格
- **配置预览**：
  - 导出标题
  - 颜色模式
  - 显示行号/讲台
- **功能特性**：
  - A4 比例适配
  - 黑白模式智能转换
  - 标签统计表
- **入口**：打开导出预览对话框（ExportPreview）
- **点击后自动关闭设置对话框**
- **数据源**：显示 `exportSettings` 预览

## 技术实现

### 组件结构
```
UnifiedSettingsDialog.vue (主对话框)
├── 顶部 Tab（全局设置/工作区设置）
├── 左侧导航（动态分类列表）
├── 右侧内容（动态加载面板组件）
└── 底部操作栏（重置/取消/保存）
```

### 数据流
- **全局设置**：通过 `useGlobalSettings` 管理，保存到 localStorage
- **云端同步**：直接使用 `useAuth` 和 `useWebDav`，不经过全局设置
- **工作区设置**：直接操作各自的 composables（`useSeatChart`、`useExportSettings` 等）

### 侧边栏 Tab 映射
- Tab 1: 文件
- Tab 2: 编辑
- Tab 3: 排位
- Tab 4: 导出

### 交互行为
- **工作区设置面板**：点击入口按钮后自动关闭设置对话框并跳转到对应功能
- **导出配置**：打开独立的导出预览对话框，而不是侧边栏
- **学生/标签管理**：打开对应的独立对话框

### 样式规范
- 主色：`#23587b`
- 卡片背景：`#f8f9fa`
- 边框：`#e2e8f0`
- 圆角：`6px`（小元素）、`8px`（卡片）
- 悬停效果：`translateX(4px)` 或 `translateY(-2px)`

## 设计原则

1. **简洁明了**：复杂功能提供入口按钮，不在设置中展开
2. **功能分离**：全局设置管理应用级配置，工作区设置管理当前工作区
3. **一致性**：云端同步面板与独立对话框功能完全一致
4. **可发现性**：通过图标、描述和统计信息帮助用户理解功能
5. **快捷访问**：提供到侧边栏或独立对话框的快速跳转
6. **自动关闭**：跳转后自动关闭设置对话框，避免界面混乱

## 用户体验

- **视觉反馈**：悬停动画、激活状态高亮
- **信息层次**：标题、描述、统计、入口按钮层次分明
- **操作引导**：使用说明框提示用户如何使用功能
- **状态显示**：实时显示当前配置和统计数据
- **流畅跳转**：点击入口按钮后自动关闭设置，直接进入目标功能
