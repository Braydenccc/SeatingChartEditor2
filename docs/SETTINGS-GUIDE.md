# 设置菜单功能说明

## 概述

统一设置对话框提供了全局设置和工作区设置两大类配置选项。

## 全局设置

全局设置会保存到 localStorage，在所有工作区中共享。

### 1. 云端同步

- **服务器地址**：WebDAV 服务器 URL
- **用户名**：WebDAV 账号
- **密码**：WebDAV 密码
- **启用自动同步**：是否自动同步到云端
- **同步间隔**：自动同步的时间间隔（分钟）

### 2. 界面偏好

- **语言**：界面语言（简体中文/English）
- **主题色**：应用主题颜色（默认 #23587b）
- **默认缩放比例**：座位图默认缩放百分比
- **启用动画效果**：是否启用界面动画
- **紧凑模式**：紧凑的界面布局

### 3. 编辑器行为

- **自动保存间隔**：自动保存的时间间隔（秒）
- **默认文件格式**：保存文件的默认格式（.sce/.json）
- **撤销历史大小**：可撤销的操作数量
- **拖拽灵敏度**：拖拽操作的灵敏度（0.5x - 2.0x）
- **双击座位行为**：双击座位时的默认操作

## 工作区设置

工作区设置仅对当前工作区生效，随工作区文件保存。

### 1. 工作区信息

显示当前工作区的统计信息：
- 学生总数
- 已分配座位数
- 空座位数
- 标签数量
- 座位总数
- 规则数量

### 2. 座位表配置

- **大组数量**：座位表的大组数（1-8）
- **讲台位置**：讲台在顶部或底部
- **座位对齐方式**：座位对齐前排或后排
- **错位距离**：相邻列的错位距离
- **各组独立配置**：每个大组的列数和行数

### 3. 自动轮换

- **启用自动轮换**：是否启用座位轮换
- **轮换周期**：轮换的时间间隔（天）
- **轮换提醒**：提前提醒的时间

### 4. 智能排位

- **最大尝试次数**：算法尝试次数（100-10000）
- **优化目标**：排位的优化策略
- **尊重区域限制**：是否考虑座位区域
- **避开空座位**：是否跳过空座位
- **保留现有分配**：是否保留已分配的座位

### 5. 导出配置

#### 通用设置
- 导出标题
- 显示标题/行号/组号/讲台
- 颜色模式（彩色/黑白/纯黑白）

#### Excel 专属
- 显示学号/边框
- 边框样式（细线/中等/粗线/虚线/点线）
- 字号设置（姓名/学号）
- 单元格宽度和行高

#### 图片导出
- 标题字号
- 姓名字号
- 边距设置

## 使用方法

1. 点击顶部导航栏的"设置"按钮打开设置对话框
2. 在顶部选择"全局设置"或"工作区设置"标签
3. 在左侧导航选择要配置的类别
4. 在右侧面板中修改设置
5. 点击"保存"按钮保存更改
6. 点击"重置"按钮恢复当前类别的默认设置
7. 点击"取消"或按 ESC 键关闭对话框

## 注意事项

- 全局设置会立即保存到 localStorage
- 工作区设置需要保存工作区文件才能持久化
- 修改座位表配置可能影响现有座位分配
- 部分功能（如自动轮换）需要在侧边栏中进一步配置
