# 云端同步设置面板更新说明

## 更新内容

云端同步设置面板已与独立的同步设置对话框保持一致，提供完整的 WebDAV 配置功能。

## 主要功能

### 1. WebDAV 配置
- **服务器地址**：输入 WebDAV 服务器 URL
- **用户名**：WebDAV 账号
- **密码/授权码**：WebDAV 密码或 Token

### 2. 自动备份模式
- **开启备份模式**：工作区列表以 SCE 云为主视角，保存/删除时自动同步至 WebDAV
- **关闭备份模式**：可单独使用 WebDAV 或 SCE 云，或将两者作为独立的写入目标

### 3. 默认读写目标
当关闭备份模式且配置了 WebDAV 时，可选择默认的云工作区读写目标：
- **SCE 云服务**：使用 Retiehe 云端存储
- **WebDAV 网盘**：使用 WebDAV 服务器

### 4. 同步状态显示
顶部信息框显示当前同步状态：
- `SCE 云 + WebDAV 双重备份`：备份模式已开启
- `WebDAV 网盘`：仅使用 WebDAV
- `SCE 云服务`：仅使用 SCE 云
- `未配置`：未配置任何同步服务

### 5. 操作按钮
- **验证并保存至账号数据库**：验证 WebDAV 连接并保存配置
- **断开/清空WebDAV**：清除 WebDAV 配置（仅在已配置时显示）

## 技术实现

### 数据源
- 直接使用 `useAuth` composable 获取和更新配置
- 不再依赖全局设置的 localStorage
- 配置保存到用户账号数据库

### 验证机制
- 保存前自动验证 WebDAV 连接
- 尝试创建 `sce_data` 目录以测试权限
- 验证失败时显示错误信息

### 状态同步
- 监听 `webdavConfig`、`backupMode`、`authType` 变化
- 自动更新面板显示内容
- 与独立同步设置对话框保持数据一致

## 与独立对话框的区别

### 相同点
- 完全相同的功能和配置选项
- 相同的验证和保存逻辑
- 一致的 UI 设计和交互

### 不同点
- 集成在统一设置对话框中，无需单独打开
- 使用设置面板的样式规范
- 操作按钮位于面板底部而非对话框底部

## 使用建议

1. **首次配置**：在统一设置中配置 WebDAV，更直观
2. **快速调整**：使用顶部菜单的"同步设置"快速访问
3. **备份模式**：推荐开启备份模式，实现双重保障
4. **定期检查**：在设置面板查看同步状态，确保配置正常
